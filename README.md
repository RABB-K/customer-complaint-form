# Customer Complaint Form

A simple, interactive web form that allows users to submit customer complaints with validation to ensure accurate and complete data entry.

## Features

- Responsive and user-friendly complaint submission form
- Input validation for all required fields (name, email, phone, complaint details, etc.)
- Real-time error messages for invalid inputs
- Clear and concise form layout
- Accessible design for better usability
- JavaScript-powered validation and form handling

## Technologies Used

- HTML5 for structure
- CSS3 for styling and layout
- JavaScript for form validation and interactivity

## How to Use

1. Open the `index.html` file in your web browser.
2. Fill in the required fields including your name, contact information, and complaint details.
3. Submit the form.
4. If any fields are invalid or missing, error messages will guide you to correct them before submission.
